###### Panera Version 1.0 ######



-------------------------
how to install
-------------------------
1. Install npm(node package manager for gulp)
2. After downloading the application, navigate to the root folder and open npm
3. Run the command "npm install"-- This will install all the required components mentioned in package.json
4. Run the command "gulp" or "npm start" -- this will start the execution of application and open the login page in browser.
